# FDF @ 42

this project is an initiating to graphic programmation and evenmentiel programmation,
he consist to print one map in "3D" witch the origine map is on text format file with extend .fdf 

#usage
```sh
./fdf [map/*.fdf]
```

#option

 - move : keyboard arrows
 - increase altitude : q
 - decrease altitude : w
 - zoom + : a
 - zoom - : s
 - quit : esc
